Ciclul ideal Diesel
  • Timpul I - admisia
  • Timpul II - compresia
  • Timpul III - arderea si detenta
  • Timpul VI - evacuarea
Raportul de compresie este:
ε = V1⁄V2 Raportul de destindere este:
ρ = V3⁄V2 Ciclul Diesel este format din:
- 2 adiabate
- 1 izocora
- 1 izobara
  • Admisia → proces izobar
    Q01 = νCp(T1 - T0)
    L01 = p1(V1 - V2)
  • Compresia → proces adiabatic
    L12 = -ΔU12 = -νCv(T2 - T1)
    Q12 = 0
  • Arderea → proces izobar
    L23 = p2(V3 - V2)
    Q23 = νCp(T3 - T2)
    - caldura primita: Q1 = Q23
  • Detenta → proces adiabatic
    L34 = -ΔU34 = -νCv(T4 - T3)
    Q34 = 0
  • Evacuarea → proces izocor(41) si proces izobar(10)
    Q41 = -νCv(T4 - T1)
    L41 = 0
    Q10 = -νCp(T1 - T0)
    L10 = p1(V2 - V1)
    - caldura cedata: Q2 = Q41
Demonstrarea formulei randamentului ciclului Diesel
η = 1 - |Qced|⁄Qabs
Qabs = Q23 = νCp(T3 - T2) = ν γR⁄γ-1 (T3 - T2)
|Qced = |Q41| = νCv(T4 - T1) = ν R⁄γ-1 (T4 - T1)
η = 1 - T4 - T1⁄γ(T3 - T2)
1-2: T1V1γ-1 = T2V2γ-1 ⇒ T1⁄T2 = (V2⁄V1)γ-1 = 1⁄εγ - 1
2-3: V2⁄T2 = V3⁄T3 ⇒ V3⁄V2 = T3⁄T2 = ρ
3-4: T3V3γ-1 = T4V1γ-1 ⇒ T4⁄T3 = (V1⁄V3)γ-1 = (γ⁄ρ)γ-1
4-1: p4⁄T4 = p1⁄T1
T1⁄T2 • T3⁄T4 = (V2⁄V1 • V1⁄V3)γ-1 = (V2⁄V3)γ-1 = 1⁄ργ-1
⇒ T1⁄T4 = (V2⁄V3)γ-1 • T2⁄T3 = (T2⁄T3)γ-1 • T2⁄T3 = (T2⁄T3)γ = 1⁄ργ
η = 1 - T4(1 - T1⁄T4) ⁄γT3(1 - T2⁄T3) = 1 - ργ - 1⁄γεγ-1(ρ - 1)
Alte caracteristici

Avantaje si dezavantaje ale motorului Diesel

   Intrucat raportul de compresie la ciclul Diesel este mai mare decat la ciclul Otto, putand ajunge chiar la 20, randamentul motorului Diesel este mai mare decat al motorului Otto.    Avantajele motorului Diesel: dezvolta putere mai mare, produce mai putine noxe, foloseste combustibil inferior, are timp de viata mai mare decat motorul Otto.
   Dezavantajele motorului Diesel: este mai greu, costul de productie este mai mare, consuma o parte din lucrul mecanic util pentru antrenarea dispozitivului care asigura pulverizarea combustibilului, necesita un sistem de preincalzire, are viteza de rotatie mica.

Marirea randamentului motorului Diesel

  • folosirea aerului comprimat la admisie
  • folosirea unui racitor intermediar pentru aerul introdus in cilindri
  • folosirea camerei de precombustie
  • combinarea motorului Diesel cu alte tipuri de motoare

Motorul Turbo-Diesel

   Putem mari puterea motorului prin redirectionarea gazelor evacuate de o turbina catre o turbina rotativa ce actioneaza o alta turbina ce introduce aer atmosferic la o presiune de 1,5 bar in cilindru.
   Efecte: micsorarea volumului si greutatii motorului; o mai buna accelerare, franare si manipulare a masinii.
