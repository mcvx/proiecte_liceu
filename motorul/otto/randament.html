Ciclul ideal Otto
  • Timpul I - admisia
  • Timpul II - compresia
  • Timpul III - arderea si detenta
  • Timpul VI - evacuarea
Raportul de compresie este:
ε = V1⁄V2 Ciclul Otto este format din:
- 2 adiabate
- 2 izocore
  • Admisia → proces izobar
    Q01 = νCp(T1 - T0)
    L01 = p1(V1 - V0)
  • Compresia → proces adiabatic
    L12 = -ΔU12 = νCv(T1 - T2)
    Q12 = 0
  • Arderea → proces izocor
    L23 = 0
    Q23 = νCv(T3 - T2)
    - caldura primita: Q1 = Q23
  • Detenta → proces adiabatic
    L34 = -ΔU34 = -νCv(T4 - T3)
    Q34 = 0
  • Evacuarea → proces izocor(41) si proces izobar(10)
    Q41 = -νCv(T1 - T4)
    L41 = 0
    Q10 = -νCp(T0 - T1)
    L10 = p1(V0 - V1)
    - caldura cedata: Q2 = Q41
Demonstrarea formulei randamentului ciclului Otto
η = 1 - |Qced|⁄Qabs
Qabs = Q23 = νCv(T3 - T2) = ν R⁄γ-1 (T3 - T2)
|Qced = |Q41| = νCv(T4 - T1) = ν R⁄γ-1 (T4 - T1)
η = 1 - T4 - T1⁄T3 - T2
1-2: T1V1γ-1 = T2V2γ-1 ⇒ T1⁄T2 = (V2⁄V1)γ-1 = 1⁄εγ - 1
2-3: p2⁄T2 = p3⁄T3
3-4: T3V2γ-1 = T4V1γ-1 ⇒ T4⁄T3 = (V2⁄V1)γ-1 = 1⁄εγ - 1
T1⁄T2 = T4⁄T3 ⇒ T1⁄T4 = T2⁄T3
η = 1 - T4(1 - T1⁄T4) ⁄T3(1 - T2⁄T3) = 1 - T4⁄T3 = 1⁄εγ - 1

ηc = 1 - T1⁄T3 > η(T1 < T4)

Bilantul energetic in ciclul real Otto

Marirea randamentului motorului Otto

   Puterea motorului Otto poate fi marita prin marirea cantitatii de amestec carburant introdus in cilindru.
   Randamentul ciclului real al motorului Otto depinde de: raportul de compresie, de natura combustibilului, de presiunea atmosferica, (la altitudine, randamentul se micsoreaza din cauza rarefierii aerului).
   Autoaprinderea combustibilului inaintea producerii scanteii de catre bujie duce la micsorarea randamentului real.    Randamentul ciclului real al motorului Otto se poate mari prin marirea raportului de compresie.
   In practica, acesta nu poate fi marit foarte mult, deoarece se produce o crestere considerabila a temperaturii in timpul compresiei, putand aparea autoaprinderea combustibilului. Valorile raportului de compresie sunt 7-12.
