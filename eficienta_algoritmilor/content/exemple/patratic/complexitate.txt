Consumatoare de operaţii este secvenţa ce implementează sortarea. Ca operaţie elementară se consideră <span class="hgl red">comparaţia (s[i] < s[j])</span>. Numărul de operaţii este egal cu <span class="hgl blue">(n - 1) + (n - 2) + (n - 3) + … + 1 = n * (n - 1) / 2 = (n<sup>2</sup> - n) / 2</span>. Am obţinut astfel o funcţie de gradul <span class="hgl red">2</span> în <span class="hgl red">n</span>. În consecinţă ordinul de complexitate al algoritmului de rezolvare este <span class="hgl blue">O(n<sup>2</sup>)</span>, algoritmul numindu-se <span class="hgl red">pătratic</span>.
